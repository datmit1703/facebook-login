# Facebook Login Page

## English

This is a Facebook login page that I created by myself after finishing my HTML5 and CSS3 studies.
The purpose of this project is to practice basic web design, layout structure, and CSS styling.

### Features

* Facebook-style login UI
* Responsive layout
* Simple and clean design
* Built with only HTML5 and CSS3

This project helped me improve my understanding of:

* HTML structure
* CSS positioning
* Flexbox layout
* Colors and typography

# Facebookログインページ
